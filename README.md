# WhyR

An R package with WhyR? Conference data sets.


## Install from GitHub
```
devtools::install_github("WhyRFoundation/WhyR")
```
##Data sets

### abstracts - WhyR? 2017

```
library(WhyR)

data(abstracts_WhyR2017)
head(abstracts_WhyR2017)

data(abstracts_WhyR2018)
head(abstracts_WhyR2018)
```
